# Contributing to NEBULAX

**Run the app:** see `HOW_TO_START_NEBULAX.md` (Django + Vite for assessment).

**Branches:** use short-lived feature branches and open a **pull request** into `main` so reviews and history stay visible for coursework.

**Secrets:** never commit `backend/.env` or real credentials.

**Coordination:** follow `docs/IT6006_Team_Contract_NEBULAX.md` (Teams + GitHub).
